Angular CLI - Instalando bootstrap

Instalando bootstrap no projeto angular

Para instalar o bootstrap basta digitar os comando no terminal, caso prefira instalar alguma versão especifica da biblioteca basta adicionar a versão da lib:
npm install bootstrap --save
npm install bootstrap@4.5.0 --save

Utilizando bootstrap com jQyery e popper.js

Para instalar a versão correta compatível com bootstrap verificar no site do bootstrap antes de instalá-la, para instalar o jQuery basta utilizar os comando a baixo:
npm install jquery --save
npm install jquery@3.5.1 --save

Você também precisará utilizar o popper.js para funcionar corretamente, ou seja o bootstrapo e dependente do jquery e js (popper) para funcionar corretamente. Assim como jQuery e preciso verificar a compatibilidade de versões no site do javascript antes de instalá-lo, para instalar digite um dos comandos:
npm install popper.js --save
npm install popper.js@1.16.0 --save

Utilizando bootstrap com a biblioteca ngx-bootstrap

Para trabalhar com angular e bootstrap não é mais necessário utilizar o jquery para isso podemos utilizar a biblioteca ngx-bootstrap (verificar versão de compatibilidade da lib). Todos os componentes do bootstrap foram recriados nessa biblioteca, então basta fazer o importe no component.ts e utilizar a tag recriada no template:
npm install ngx-bootstrap --save

Referenciando as bibliotecas

Agora bastar referenciar as bibliotecas no angular.json. Obs: os arquivos tem quer ser colocados exatamente nessa sequencia.

Para poder utilizar o css do bootstrap deverá ser referenciado no no style que fica dentro do arquivo angular.json:
“node_modules/bootstrap/dist/css/bootstrap.css”

Em scripts coloque:
“node_modules/jquery/dist/jquery.js”,
“node_modules/popper.js/dist/umd/popper.js”,
“node_modules/bootstrap/dist/js/bootstrap.js”

Salve o arquivo e reinicie o servidor Angular, para verificar as versões das bibliotecas instaladas basta abrir o arquivo package.json
